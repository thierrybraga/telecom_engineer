# Algoritmos e Estruturas de Dados II

## 1. Ementa Expandida
Estudo de estruturas de dados dinâmicas, árvores, grafos e algoritmos avançados. Desenvolvimento de habilidades para modelagem e resolução de problemas complexos em computação.

## 2. Blocos Teóricos

### 2.1 Estruturas Dinâmicas
- Listas encadeadas: simples, duplas, circulares
- Pilhas e filas: implementações e aplicações
- Deques (filas de dupla extremidade)
- Tabelas hash

### 2.2 Árvores
- Conceitos fundamentais: raiz, nós, folhas, altura
- Árvores binárias: percursos (in-order, pre-order, post-order)
- Árvores binárias de busca (BST)
- Árvores balanceadas: AVL, Rubro-Negra
- Heaps e filas de prioridade

### 2.3 Árvores Avançadas
- Árvores B e B+
- Tries (árvores de prefixos)
- Árvores de segmento
- Árvores de Fenwick (Binary Indexed Tree)

### 2.4 Grafos
- Representação: matriz de adjacência, lista de adjacência
- Percursos: BFS (Busca em Largura), DFS (Busca em Profundidade)
- Conectividade e componentes
- Ordenação topológica

### 2.5 Algoritmos em Grafos
- Caminho mínimo: Dijkstra, Bellman-Ford, Floyd-Warshall
- Árvore geradora mínima: Kruskal, Prim
- Fluxo máximo: Ford-Fulkerson, Edmonds-Karp
- Emparelhamento máximo

### 2.6 Algoritmos Avançados
- Programação dinâmica
- Algoritmos gulosos
- Backtracking
- Branch and bound

## 3. Aplicações em Telecom

### 3.1 Redes de Comunicação
- Modelagem de topologias de rede como grafos
- Roteamento: algoritmos de caminho mínimo
- Spanning tree protocol
- Detecção de ciclos em redes

### 3.2 Processamento de Dados
- Estruturas para roteamento de pacotes
- Tabelas de encaminhamento
- Balanceamento de carga
- Cache e memoização

## 4. Prática/Laboratório
- Implementação de estruturas de dados em C/C++
- Simulação de algoritmos de roteamento
- Análise de complexidade empírica
- Resolução de problemas competitivos
